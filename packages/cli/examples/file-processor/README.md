# File Processor CLI Example

Advanced file processing CLI demonstrating filesystem operations, streaming, and batch processing.

## Features

- Line counting
- Word counting  
- Text transformations
- Batch processing
- Progress tracking

## Usage

```bash
npm install
npm run dev process README.md --operation count-lines
npm run dev process README.md --operation word-count
npm run dev process README.md --operation uppercase
```
