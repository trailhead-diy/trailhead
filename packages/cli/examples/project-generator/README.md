# Project Generator CLI Example

Demonstrates project scaffolding and template generation using @esteban-url/trailhead-cli.

## Features

- Template-based project generation
- Interactive prompts
- File/directory creation
- Configuration customization

## Usage

```bash
npm install
npm run dev generate --name my-project --template basic
npm run dev generate --name my-cli --template cli
```
