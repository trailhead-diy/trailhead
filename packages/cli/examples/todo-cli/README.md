# Todo CLI Example

A complete todo application demonstrating state management, persistence, and CRUD operations with @esteban-url/trailhead-cli.

## Features

- Add/remove todos
- Mark as complete
- Persistent storage
- List filtering
- Import/export

## Usage

```bash
npm install
npm run dev add "Learn @esteban-url/trailhead-cli"
npm run dev list
npm run dev complete 1
```
