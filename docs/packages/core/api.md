# Core API Reference

The @esteban-url/core package provides foundational utilities for the Trailhead system.

## Installation

```bash
npm install @esteban-url/core
```

## Key Features

- **Result Types**: Functional error handling with Result type
- **Error Management**: Structured error creation and handling
- **Functional Utilities**: Composition helpers and functional programming patterns
- **Type Guards**: Runtime type checking utilities

See the [TypeScript definitions](https://github.com/esteban-url/trailhead/tree/main/packages/core/src) in the source code for detailed API documentation.
